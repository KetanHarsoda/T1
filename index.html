TV

Kodi repository for add-ons to stream Indian OTT Content

Project maintained by ketanharsodaibl Hosted on GitHub Pages


Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://ketanharsodaibl.github.io/ (Give it the name "TV Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "TV Repository", and install jio-tv-2.0.25.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "TV Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
